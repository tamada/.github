## Overview

This repository contains default community health files for GitHub repositories. When placed in a `.github` repository within an organization or user account, these files will be used as defaults across all repositories that don't have their own versions.

## Contents

- **[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)** - Community code of conduct based on Contributor Covenant v2.1 [Japanese version](CODE_OF_CONDUCT.ja.md)
- **[CONTRIBUTING.md](CONTRIBUTING.md)** - Guidelines for contributing to projects [Japanese version](CONTRIBUTING.ja.md)

## Usage

### For Organizations

1. Create a public repository named `.github` in your organization
2. Add these community health files to the repository
3. These files will automatically apply to all repositories in your organization that don't have their own versions

### For Individual Users

1. Create a public repository named `.github` in your user account
2. Add these community health files to the repository
3. These files will serve as defaults for all your public repositories

## Customization

Feel free to customize these templates to match your project's needs:

- Update contact information and enforcement policies in CODE_OF_CONDUCT.md
- Modify development setup and requirements in CONTRIBUTING.md
- Add organization-specific guidelines and standards
